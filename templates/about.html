About This Site

Who I Am

I'm Finlay Anderson, a Computer Science student interested in software engineering, problem solving, and building practical systems and just engineering/technology in general. I enjoy taking core computing concepts and turning them into small working applications rather than leaving them as theory.

Why I Built This

This website is designed as a lightweight portfolio demonstrating my understanding of backend development fundamentals using Flask. Instead of a purely visual project, I wanted to build something that shows logic, validation, and clear reasoning.

The binary converter is intentionally simplistic - the focus is correctness, clarity, and structured code rather than complexity. Each conversion shows the working steps to reflect how the algorithm actually operates.

What It Demonstrates

  • Flask routing and templating
  • Form handling and validation
  • Separation of frontend and backend logic
  • Readable, maintainable code structure
